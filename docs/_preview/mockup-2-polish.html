Agent Sessions
macOS · Open Source ★ 128

Session management for Codex, Claude, OpenCode, Cursor, Copilot, Pi and Gemini

Agent Sessions is a local-first Mac app for session history across CLI tools, desktop apps, and app-bundled agent commands. Search local Codex, Claude, OpenCode, Cursor, Copilot, Pi, Gemini, Hermes, and OpenClaw histories, inspect transcripts and images, then resume supported CLI sessions in Terminal.app, iTerm2, or Warp.

Quota Meter and Agent Cockpit

The Quota Meter is a small always-on Codex and Claude usage window for MacBook users who want 5h and weekly limits visible without spending menu-bar space. Freshness-aware predictions show projected run-out ETAs before the final alert window, while Agent Cockpit provides grouped active and waiting visibility, quick focus jumps, and compact/full modes.

Resume any session

Right-click a supported CLI session to copy the exact resume command or open it directly in Terminal.app, iTerm2, or Warp.

Saved sessions

Saved Sessions window for managing archived sessions. Delete, reveal, and diagnose with archive status tooltips.

Usage tracking

Track Codex and Claude CLI usage in-app, in the menu bar, or in the Quota Meter. Shared notifications warn when limits are approaching, exhausted, or projected to run out.
