# Marketing Screenshot Directory

Screenshots follow the `version-description` naming pattern. A `dark` segment indicates Dark Mode; otherwise the screenshot is treated as Light Mode. For the main Sessions window, `v` indicates vertical layout, `h` indicates horizontal layout, and no layout segment defaults to vertical.

All screenshot files have image extensions and use lowercase kebab-case filenames. Large screenshots were resized for web use with a maximum long edge of 1600 px while preserving aspect ratio; already-compact captures were kept compact.

Visual review notes: no obvious secrets, credentials, or offensive text were visible in the screenshots during a quick contact-sheet review. Some screenshots show local paths, UUID-like identifiers, terminal output, issue names, and session titles as expected for product screenshots.
